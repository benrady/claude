## Developer Tasks

All common developer tasks are automated via `make`. Run `make help` to see available targets. Dependencies (including the virtual environment) are managed automatically. You do not need to run `uv sync` or activate the venv manually — Make targets handle this.

### Examples

* Running tests -- `make test`
* Formatting code -- `make format`
* Linting code -- `make lint`
* Running pre-commit hooks `make pre-commit`
